# Budget TRACKER
Here is a budget tracker to track your spending!

## Description
This budget tracker uses indexedDB for offline functionality so you can track your spending at any time. 

## Technology
Key technologies used in this project:
- HTML
- CSS
- JS
- indexDB
- Bootstrap
- express.js
- morgan
- mongoose
- compression
